# Power BI Dashboard Project

## Overview
This project is an interactive Power BI dashboard designed for business reporting and KPI analysis.

## Features
- Interactive slicers
- KPI cards
- Trend analysis
- Comparative charts
- Matrix reporting
- Dynamic filtering

## Tools Used
- Power BI
- DAX
- Power Query

## Key Insights
- Business performance tracking
- Category comparison
- Trend monitoring
- KPI visualization

## Project Structure
- `pbix/` → Power BI file
- `screenshots/` → Dashboard images
- `dataset/` → Source data
- `docs/` → Documentation
